Detailed Description

Individual Array Buffers Used for I2C Communication Each Corresponds to an Inno Pro I2C register.

Individual Register Values that needs to be configured.

These Array Buffers needs to be filled with LSB and MSB values These are used directly for Writing values to Inno Pro via I2C

These values are Initialize with Inno Pro Default Values

Buffer[0] - LSB Buffer[1] - MSB

After these values have been set, it is converted to appropriate MSB and LSB needed for Filling Up the Inno Pro I2C buffers
